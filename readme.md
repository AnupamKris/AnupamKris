### 🎙️ [Windows Voice Assistant](https://github.com/anupamkris/winassist)
A low-latency, hands-free Windows Copilot alternative built using **Google's Multimodal Live API**, **Tauri**, and **React**. It executes native system-level tasks and integrates with the Gmail API for advanced voice-composed email management.

### 🌿 [PyFlow](https://github.com/anupamkris/pyvis)
A visual scripting alternative version of Python built completely from scratch using **ReactJS** and advanced DOM manipulation to allow users to construct logical backend flows visually.

### 📁 [quick-file-server](https://github.com/anupamkris/quick-file-server)
A portable, single-command file manager and editor built using **FastAPI** and **React (Vite + Tailwind + Shadcn UI)**. Shipped as a self-contained Python wheel, it lets developers instantly spin up an interactive file browser and code editor in any directory via `uvx quick-file-server` with zero local configuration.

### 🎩 [Monopoly Banker](https://github.com/AnupamKris/monopoly-banker) | [Web](https://monopoly.anupamkris.dev)
A real-time, digital banking companion for physical Monopoly board games built using React and hosted on Vercel. It replaces physical cash with a seamless digital ledger, allowing players to instantly transfer funds, manage properties, track transaction histories, and eliminate banker calculation errors.

### 📊 [Data2Insights](https://github.com/AnupamKris/data2insights)
An interactive AI-powered data chat application built with Vue 3 and Vite. It allows users to upload dataset files, ask natural language questions about their data, and instantly receive text insights and generated data visualization charts.

### 📱 [DreamDeck](https://github.com/AnupamKris/DreamDeck)
A custom smart control deck / remote interface (inspired by Elgato Stream Deck) built as a hybrid mobile app using Vue 3 and Capacitor. It communicates with a local desktop socket server (Socket.io) to control monitor brightness, manage granular process volumes, switch OBS scenes, and trigger application shortcuts with haptic feedback.
