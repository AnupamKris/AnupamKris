### 🎙️ [Windows Voice Assistant](https://github.com/anupamkris/winassist)
A low-latency, hands-free Windows Copilot alternative built using **Google's Multimodal Live API**, **Tauri**, and **React**. It executes native system-level tasks and integrates with the Gmail API for advanced voice-composed email management

### 🌿 [PyFlow](https://github.com/anupamkris/pyvis)
A visual scripting alternative version of Python built completely from scratch using **ReactJS** and advanced DOM manipulation to allow users to construct logical backend flows visually

### 🎩 [Monopoly Banker](https://github.com/AnupamKris/monopoly-banker) | [Web](https://monopoly.anupamkris.dev)
A real-time, digital banking companion for physical Monopoly board games built using React and hosted on Vercel. It replaces physical cash with a seamless digital ledger, allowing players to instantly transfer funds, manage properties, track transaction histories, and eliminate banker calculation errors.

> **quick-file-server** is a developer-friendly, zero-install tool to instantly spin up an interactive file browser and editor in any directory. It combines the speed of a FastAPI backend with a modern, responsive React web interface (powered by TailwindCSS and Shadcn UI), packaged into a single Python wheel. Simply execute `uvx quick-file-server` to run it instantly without configuring Node or Python virtual environments.
